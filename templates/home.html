AES Encryption and Decryption
Online Tool

Advanced Encryption Standard(AES) is a symmetric encryption algorithm. AES is the industry standard as of now as it allows 128 bit, 192 bit and 256 bit encryption.Symmetric encryption is very fast as compared to asymmetric encryption and are used in systems such as database system. Following is an online tool to generate AES encrypted password and decrypt AES encrypted password. It provides two mode of encryption and decryption ECB and CBC mode. For more info on AES encryption visit this explanation on AES Encryption..

USER GUIDE


Any plain-text input or output that you enter or we generate is not stored on this site, this tool is provided via an HTTPS URL to ensure that text cannot be stolen. For encryption, you can either enter the plain text and password that you want to encrypt. The input plain text will be divided into blocks and each block will be encrypted with the password provided and hence identical plain text blocks are encrypted into identical cipher text blocks. The AES algorithm has a 128-bit block size, regardless of whether you password length is 256, 192 or 128 bits. When a symmetric cipher mode requires an IV, the length of the IV must be equal to the block size of the cipher. AES provides 128 bit, 192 bit and 256 bit of password size for encryption. The password must be of 16 bits long and 24 and 32 bits for 192 and 256 bits of size. Now you can enter the password accordingly.The encrypted text will be base64 encoded.
